#### vuepress-blog-template
### 介绍
基于[vuepress](https://www.vuepress.cn/)静态网站生成器

### 安装
```sh
npm install
```

### 启动
``` sh
npm run dev
```

默认：并运行在**http://localhost:5556/** 
可在`docs/.vuepress/config.js`找到`port`修改


### 打包
``` sh
npm run build
```
